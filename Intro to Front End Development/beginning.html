  • 3. How does Internet Work?

    • Internet: An interconnection of thousands of computers across the globe
    • Client-Server Model: Whenever a computer(client) makes a request to access a webpage (for instance: www.google.com), it sends request to ISP (the internet provider company that allows you to use/access internet), which locates the domain name in DNS (Domain Name Server, acts as a phonebook), which then redirects the request to Server(A Large SuperComputer that acts as the library of webpages) which then returns the requested data to client computer.

  • 4. How Website Works

    1. Browser: It is a window/application that helps us to use Internet. For Example: Chrome, Edge, FireFox etc.
      Whenever a client makes request to server for a website, the server sends a set of files consisting of HTML, CSS and JavaScript.
      1. HTML: Hypertext Markup Language, serves as structuring the website. i.e: The Text we see on web pages.
      2. CSS: Cascading Style Sheet, serves in styling the webpages and make websites attractive. i.e: Apply Fonts to text or colors to buttons etc.
      3. JavaScript: A Programming Language, that serves in the functionality of the website. i.e: making the Buttons or other Components to respond the way we want.

Day-3: Covering "Introduction to HTML"

  • 1. Introduction to HTML

    • Working of Browsers: Whenever a client sends request, the server sends a collection of HTML, CSS & JS Files; which are used to render the display of interface to be used by the user
    • What is HTML? HTML stands for Hypertext Markup Language. The word Markup means that it is used to structure the documents or more specifically, webpages.
    • HTML Tags: HTML Tags are used to structure the documents. They are enclosed in angle brackets. For Example: <h1> </h1> is a tag used to define a heading.
    • HTML Attributes: HTML Attributes are used to provide additional information about the elements. For Example: <img src="image.jpg" alt="Image"> is a tag used to define an image. Here, src and alt are attributes.
    • HTML Elements: HTML Elements are the building blocks of HTML Pages. For Example: <h1> </h1> is an element used to define a heading.

  • 2. The Anatomy of HTML Tag

    • Opening Tag: It is the first part of the tag. For Example: <h1>
    • Closing Tag: It is the second part of the tag. For Example: </h1>
    • Content: It is the content of the tag. For Example: <h1>Hello World</h1>
    • Self Closing Tag: It is the tag that does not have a closing tag. For Example: <br>
